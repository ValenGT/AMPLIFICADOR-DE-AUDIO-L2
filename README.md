# Amplificador Analógico – Diseño y Análisis

Este repositorio contiene el **diseño, análisis y documentación de un amplificador analógico de tres etapas**, desarrollado como parte del estudio de **electrónica analógica**.

El proyecto incluye:

- Cálculos de diseño del amplificador
- Scripts utilizados para automatizar cálculos
- Informe técnico del proyecto
- Presentación del trabajo
